## 简介
本文件系统性阐述 Vben Admin 的暗色主题体系，涵盖设计原理、实现机制、色彩映射规则、自动切换逻辑、组件适配策略、系统检测与用户偏好设置、测试与质量保障，以及设计规范与实现示例路径。目标是帮助开发者与设计师在不同 UI 框架与业务场景中稳定地落地暗色主题，并确保可读性与视觉舒适度。

## 详细组件分析

### 偏好管理器与自动切换
- 初始化与合并：从缓存加载用户偏好，与默认配置与覆盖项合并，立即应用。
- 主题更新：当 theme 字段变更时，调用 CSS 变量更新器；当 app.colorGrayMode 或 app.colorWeakMode 变更时，切换页面颜色模式类名。
- 自动模式监听：监听系统 prefers-color-scheme 变化，仅在 mode 为 auto 时跟随系统并恢复 auto 状态，避免用户手动切换被覆盖。

```mermaid
flowchart TD
Start(["初始化偏好"]) --> LoadCache["加载缓存偏好"]
LoadCache --> Merge["合并默认与覆盖项"]
Merge --> Apply["立即应用主题与颜色模式"]
Apply --> WatchSys["监听系统主题变化"]
WatchSys --> ModeAuto{"当前模式=auto?"}
ModeAuto --> |是| FollowSys["跟随系统并恢复auto"]
ModeAuto --> |否| End(["结束"])
```

### CSS 变量与类名更新
- 模式判定：支持 light/dark/auto；auto 时通过 matchMedia 判断系统偏好。
- 根节点类名：根据模式为 documentElement 添加/移除 "dark" 类，驱动 CSS 选择器。
- 主题类型：通过 dataset.theme 切换内置主题类型，配合设计令牌按类型应用变量。
- 颜色映射：将主题主色与语义色映射到统一 CSS 变量，再由通用工具写入根节点。
- 圆角与字号：同步更新 --radius 与 --font-size-base 等基础变量。

```mermaid
flowchart TD
Enter(["updateCSSVariables"]) --> CheckMode["检查 theme.mode"]
CheckMode --> IsAuto{"mode=auto?"}
IsAuto --> |是| SysDark["matchMedia判断系统深色"]
IsAuto --> |否| SetDark["直接按light/dark"]
SysDark --> SetDark
SetDark --> ToggleClass["为<html>添加/移除'dark'类"]
ToggleClass --> ThemeType["设置dataset.theme类型"]
ThemeType --> Preset["解析内置主题预设"]
Preset --> PrimaryColor["确定主色(考虑暗色优先)"]
PrimaryColor --> ColorVars["生成语义色变量映射"]
ColorVars --> WriteCSS["写入CSS变量与通用变量"]
WriteCSS --> RadiusFont["更新圆角与字号变量"]
RadiusFont --> Exit(["完成"])
```

## 性能考量
- 防抖持久化：偏好更新采用防抖保存，降低频繁写入缓存的开销。
- 条件更新：仅在主题或颜色模式相关字段变更时触发 CSS 变量更新，避免无谓重绘。
